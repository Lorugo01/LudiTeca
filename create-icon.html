Criar Ícone para UniverseTeca CMS

Esta ferramenta irá criar um ícone simples com as letras "UT" que você pode usar para o atalho do projeto.

Como usar este ícone:

  1. Clique em "Gerar Ícone"
  2. Clique em "Baixar PNG"
  3. Use um conversor online para transformar o PNG em ICO: Convertio ou PNG to ICO
  4. Crie um atalho para o arquivo start-universeteca.bat
  5. Clique com botão direito no atalho, escolha "Propriedades"
  6. Clique em "Alterar ícone" e selecione o arquivo ICO que você criou
