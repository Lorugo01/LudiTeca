# UniverseTeca CMS

Sistema de Gerenciamento de Conteúdo para a Biblioteca Digital UniverseTeca.

## Inicialização Rápida

Para iniciar o sistema com apenas um clique:

1. **Clique duplo** no arquivo `start-universeteca.bat` na pasta do projeto
2. O servidor iniciará automaticamente e um navegador será aberto no endereço http://localhost:3000
3. Faça login com suas credenciais

## Requisitos do Sistema

- Node.js 14.x ou superior
- Navegador moderno (Chrome, Firefox, Edge)
- Conexão com internet (para autenticação com Supabase)

## Funcionalidades

- Gerenciamento de livros digitais
- Cadastro e manutenção de autores
- Organização por categorias
- Perfil de usuário personalizado
- Upload e gerenciamento de mídias

## Configuração Manual

Se preferir iniciar manualmente:

1. Abra um terminal na pasta do projeto
2. Execute `npm install` (somente na primeira vez)
3. Execute `npm run dev`
4. Acesse http://localhost:3000 no navegador

## Suporte

Em caso de problemas:

1. Verifique se o Node.js está instalado corretamente
2. Confirme que as variáveis de ambiente estão configuradas
3. Verifique a conexão com internet

Para mais informações ou ajuda, contate o suporte técnico.
